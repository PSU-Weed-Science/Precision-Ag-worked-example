In this repository is a .RMD file that will take you through some basics of working with yield data over a whole field.

After that, we'll do a quick example on using a multi-year data to create yield stability zones. 
If you need to revisit anything, remember to use the outline to jump to your sections, but always be careful to make sure your objects are what you think they are!
